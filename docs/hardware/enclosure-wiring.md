# Enclosure Wiring

## Enclosure

The tested enclosure is an outdoor lockable electrical box from Amazon ASIN `B0CSJXTLGZ`.

## DIN rail power supply

The build uses an Ideal Power `56YSD60S-1204500` 12 V DIN rail power supply.

## DIN terminal blocks

The enclosure uses Dinkle DK2.5N DIN rail terminal blocks.

| Item | Value |
| --- | --- |
| Series | Dinkle DK2.5N |
| Wire range | 12 to 22 AWG |
| Colors | Various colors |
| Use | Enclosure wiring and distribution |

## Cable glands

The build uses TE Connectivity ENTRELEC `1SNG601161R0000` cable glands to protect cable entries into the enclosure.

## Front panel

The front panel includes:

- breakaway manual/auto switch section
- panel LEDs for Valve 1 and Valve 2
- optional local open/close control through the breakaway board
